Welcome to Josie's D&D Resource site!
This website is a simple reference tool for some aspects of our D&D game.

You can check out some maps to familiarize yourself with the layout of the world, check the scheduling calendar, look back on the notes for previous sessions, learn about the different races that inhabit the world, read some in-world books that may offer some insight to the history of the world, or learn about the pantheon.

Let me know if you can think of anything else you'd like to see here!
